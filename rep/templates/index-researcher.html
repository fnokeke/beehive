Behavior Change Framework

study human behaviors and deploy interventions in real-world settings using mobile devices.



About

Beehive makes it easy for researchers to capture participant/patient reported outcomes using mobile devices.




Researchers

Customize Beehive interface to fit the specific needs for your research project using our web-based manager. Features:

  • create onboarding instructions.
  • select mobile datastreams to capture
  • design interventions tailored to each user
  • monitor participants' progress through live dashboard
  • download data for extensive offline analysis

Currently available to Cornell Tech, UMich, and OSU research teams. Coming soon to other institutions.


Participants

Onboarding participants should:

  1. Install Beehive app(s)
  2. Enter experiment code
  3. Use their phones as they normally do
 App Store Play Store

Features

Mobile Sensing

Using embeded sensors, uncover users' daily habits without interrupting them.

Digital Nudges

Send push notifications, phone vibrations, and more to participants.

User Contexts

Deploy interventions based on user locations, time, phone usage habits.




Technology

Beehive leverages the existing frameworks such as ResearchKit and ResearchStack to capture user behaviors. Using Beehive, researchers can design diverse experiments with different mobile interventions for groups of participants.
