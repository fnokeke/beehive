About

Beehive makes it easy for researchers to capture participant/patient reported outcomes using mobile devices.




Researchers

Customize Beehive interface to fit the specific needs for your research project using our web-based manager. Features:

  • add project participants
  • create tasks/surveys using Beehive
  • create active tasks
  • schedule when and how often tasks should appear
  • send secure messages to individual participants
  • send announcements to all participants
  • create information screens
  • create a custom theme

Participant data is synchronized to your Beehive project. Available to Cornell research teams now. Coming soon to other institutions.


Participants

If you are participating in a research project that uses Beehive:

  1. Install the app
  2. Open the app and tap the Join Project button
  3. Scan the QR code that your project coordinator provided

Features

Security

Beehive protects participant data by requiring a 4-digit passcode to unlock the app. Data are stored in an encrypted database on the device.

Communication

Send announcements to all project participants or a message to an individual participant.

Scheduling

Flexible scheduling for one-time tasks, repeating tasks, and permanent tasks.

Results

Synchronized to the researcher's Research Suite project.




Technology

Beehive leverages BeeHive, ResearchKit, and ResearchStack to capture participant/patient reported outcomes via mobile devices. Beehive is used to define tasks/instruments/surveys to be completed by participants. Beehive translates Beehive task metadata into a structure compatible with ResearchKit and ResearchStack. When a project participant completes a task, Beehive converts the results into a format compatible with Beehive before synchronizing back to the Beehive project.
